# 🤖 AI Question Answering Bot (Gemini)

A simple text-based AI Question Answering bot built using **Google Gemini API**.

## 🚀 Features
- Ask any question in simple English
- Beginner-friendly CLI interface
- Uses Gemini 2.5 Flash (free tier)

## 🛠️ Tech Stack
- Python
- Google Gemini API
- google-genai SDK

## ▶️ How to Run

```bash
git clone https://github.com/your-username/genai-qa-bot.git
cd genai-qa-bot
pip install -r requirements.txt
